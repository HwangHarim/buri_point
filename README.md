## 🔨 개발 환경
- JAVA 17
- IDE: IntelliJ
- Framework: SpringBoot 3.0
- DB: MySQL, Redis
- ORM: JPA
- Thymeleaf
- Rest Docs

<br>

## 📌 기능 요구사항
1. 회원 가입 - 이름, 이메일, 비밀번호, 핸드폰 번호, 핀 번호, 보유 포인트
2. 로그인, 로그아웃
3. 회원 탈퇴
4. 부리 포인트 결제
5. 1/N 정산하기
6. 포인트 충전 및 이체
7. 결제(이체) 내역 조회
